# E-Commerce Data Warehouse

## Overview

Designed a Star Schema-based data warehouse for e-commerce analytics using PostgreSQL and SQL.

## Technologies Used

* PostgreSQL
* SQL
* Data Warehousing
* Star Schema

## Schema Design

* Fact Table: fact_sales
* Dimension Tables:

  * dim_customer
  * dim_product
  * dim_date

## Features

* Star Schema implementation
* Fact and Dimension table design
* SQL JOIN queries
* Revenue and sales analytics

## Sample Reports

* Customer-wise Sales Analysis
* Product-wise Revenue Analysis
* Category-wise Revenue Reporting
